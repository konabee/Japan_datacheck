# Japan_datacheck

This repository is used by Polina Dorfman and Chia Liu to conduct data quality checks on suicide statistics in Japan, using data from both the <a href="https://www.mortality.org/Data/HCDHMD" target="_blank" rel="noopener noreferrer">Human Mortality Database</a> and <a href="https://www.e-stat.go.jp/en/stat-search/files?page=1&amp;layout=datalist&amp;toukei=00450011&amp;tstat=000001028897&amp;cycle=7&amp;tclass1=000001053058&amp;tclass2=000001053061&amp;tclass3=000001053065&amp;tclass4val=0&amp;metadata=1&amp;data=1" target="_blank" rel="noopener noreferrer">Japanese government data</a>.
